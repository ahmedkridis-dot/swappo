# Swappo Translation Keys — Complete Reference

This is the master list of all translation keys used across Swappo. Check this file before creating new keys.

## Key Naming Rules

- Use `snake_case` for key names
- Use dot-notation: `page.section.element`
- Keep keys descriptive but concise
- Group by page/feature, not by UI element type
